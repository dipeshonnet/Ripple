# Mobile Demo Instructions

## Best option: Netlify static hosting

1. Open Netlify Drop: https://app.netlify.com/drop
2. Upload the app folder contents so the site root directly contains:
   - index.html
   - styles.css
   - data.js
   - app-core.js
   - app-views-agent.js
   - app-views-lead-mgr.js
   - app-modals.js
3. Open the Netlify URL on iPhone/Android.
4. Join Teams from laptop for audio and speaker notes. Join the same meeting from phone muted and share phone screen.

## Desktop simulator fallback

Run `launch_desktop.bat`, open Chrome DevTools, toggle mobile viewport, and share the browser window in Teams.

## Notes

The full app is one responsive static app. Do not use iPhone file preview for the full demo; it can distort CSS/JS rendering.
